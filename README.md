# taro

## Requirements

| Tool    | Version    |
| ------- | ---------- |
| Node.js | >= 10.11.0 |
| npm     | >= 6.4.1   |

## Setup

```
$ npm install
```

## Build

```
$ npm run build
```

## Lint

```
$ npm run tslint
```

## Prettier

```
$ npm run prettier
```

## Watch
```
$ npm run watch
```
